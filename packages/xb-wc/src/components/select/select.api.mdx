# select

Select help users pick one or more of several options, or search for an option when it's not available.

## Usage

```jsx
import '@welingtonms/xb-wc/dist/select';
```

### Avoid mixing static options with datasources

- Static options value will always be either a `string` or `string[]`, for multiple selection
- Datasource-based options will depend on the objects they return; the select will give back the raw object that represent the selection option(s)
- Alternatively, you can wrap your static options in a sync datasource

## Properties

| Property   | type                                         | Description                                        |
| :--------- | :------------------------------------------- | :------------------------------------------------- |
| `disabled` | `Boolean`                                    | **Optional**. Should the button be disabled.       |
| `emphasis` | `'text'` &#124; `'ghost'` &#124; `'flat'`    | **Optional**. Emphasis variant. Default `'ghost'`. |
| `size`     | `'small'` &#124; `'medium'` &#124; `'large'` | **Optional**. Size variant. Default `'small'`.     |

## Named Slots

```html
<xb-button>
	<span slot="leading">Leading</span>
	Content
	<span slot="trailing">Trailing</span>
</xb-button>
```

| Slot Name  | Description                                                   |
| :--------- | :------------------------------------------------------------ |
| `leading`  | **Optional**. Element to be rendered before the main content. |
| `trailing` | **Optional**. Element to be rendered after the main content.  |
