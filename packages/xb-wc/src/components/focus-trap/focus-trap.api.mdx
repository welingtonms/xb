# Focus Trap

Texts convey information necessary to guide users to perform any actions in an application.

## Usage

```js
import '@welingtonms/xb-wc/dist/text';
```

```html
<xb-text>The quick brown fox jumps over the lazy dog</xb-text>
```

## Properties

| Property  | type                                                                                                                                                                                                                        | Description                                                       |
| :-------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :---------------------------------------------------------------- |
| `variant` | `'h-1'` &#124; `'h-2'` &#124; `'h-3'` &#124; `'h-4'` &#124; `'h-5'` &#124; `'h-6'` &#124; `'subtitle-1'` &#124; `'subtitle-2'` &#124; `'body-1'` &#124; `'body-2'` &#124; `'button'` &#124; `'caption'` &#124; `'overline'` | **Optional**. Typography style to be applied. Default `'body-1'`. |
