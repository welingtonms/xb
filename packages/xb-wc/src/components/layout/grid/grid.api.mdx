## Properties

| Property      | type                                                      | Description                                           |
| :------------ | :-------------------------------------------------------- | :---------------------------------------------------- |
| `paddingless` | `Boolean` &#124; `DirectionPropType[,DirectionPropType]*` | **Optional**. Which padding side should be supressed. |
